Main script, to make an .exe out of, is: `serial_communication_for_EXE.py`

All pip installs:

// Python controller's audio commands:
pip install comtypes
pip install https://github.com/AndreMiras/pycaw/archive/master.zip

// for Arduino communication:
pip install pyserial

// for notifications:
pip install win10toast
pip install gTTS
pip install playsound
